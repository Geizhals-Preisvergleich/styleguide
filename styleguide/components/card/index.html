Groups of UI elements which belong to the same functionality can be grouped in Cards. Cards help to communicate the relation between UI elements and organizes areas of the UI into distinctive groups.

Modern Cards ðŸ”—

Currently, Cards are displayed as a box with a border of "grey-lighter-40" (see Colors), white background and padding of 1rem:

.mycard {
  background: #fff;
  border: 1px solid @grey-lighter-40;
  padding: 1rem;
}

This type of card is used on these pages, among others:

  • Search Results (SERP)
  • Deals
  • Variant
  • Product Ratings
