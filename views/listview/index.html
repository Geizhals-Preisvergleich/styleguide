Listview is a versatile component to display a list of products. It displays the most important data in a compact way. The component can usually be opened to show more details.

It is currently used on these pages, in slightly different ways:

  • Searchresults
  • Variant (mobile)
  • Variant offers
  • Dealspage

Dependencies:

  • Dots Menu (dots-dropdown)
  • CSS Stars (css_stars)
  • Tooltips (tooltip)
