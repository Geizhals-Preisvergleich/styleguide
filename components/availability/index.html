The Availability component is used to communicate to the user if an item is available at the moment.

It comes in two forms:

  • as a small circle, see Dot
  • as a vertical box, see Box

Dependencies:

  • Tooltips
Going forward we aim to use the Availability Dot exclusively and gradually replace the Availability Box.
