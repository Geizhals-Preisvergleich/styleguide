Outlined versus filled icons ðŸ”—

Use both filled and outlined icons depending the required context. For instance, a filled icon could be used to indicate that something is "on", while an outlined icon could indicate that something is "off".

Do's ðŸ”—

  • Add a label to the icon where possible. Otherwise users might not know what we intend to communicate with this particular icon.
  • If a label is not possible due to layout constraints, consider adding a tooltip.
  • States of an icon (like active, inactive ...) should be communicated clearly, ideally along with the corresponding label. This ensures that visually impaired users understand the meaning of the icon.
  • Pay attention to proper sizing and alignment of multiple icons to maintain visual balance. Certain icons appear visually out of balance or have different intrinsic sizes.
Always add an Alt text and at least, if the image has no label, a 
title
attribute.

Dont's ðŸ”—

  • Aim to use existing icons instead of creating new files.
